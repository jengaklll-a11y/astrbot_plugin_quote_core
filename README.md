# 语录(Core)

一个 AstrBot 插件（以下示例假设指令前缀为 `/`）：

- **上传语录**：发送 `/上传 <内容>`。
  - 支持指定归属人：`/上传 内容 @某人`。
  - 不 @ 则默认归属上传者。
  - *注意：目前 Core 版本仅支持文本语录。*
  
- **随机语录**：发送 `/语录`。
  - 可选：`/语录 @某人` 仅随机该用户的语录。
  - 可选：`/语录 5 @某人` 随机抽取该用户的 5 条语录并生成合集长图。
  - 渲染：使用 HTML 渲染精美语录卡片（强制显示头像，默认使用 qlogo）。

- **删除语录**：在“回复机器人刚发送的随机语录消息”时发送 `/删除`，即可删除该语录。

- **戳一戳触发**：在支持 OneBot v11/Napcat 的 QQ 平台中，对机器人使用“戳一戳”（Poke）时，将自动随机发送一条语录（需开启配置）。

## 配置

- `poke_enabled`：是否启用“戳一戳触发随机语录”功能，布尔值，默认 `true`。
- `admin_only`：是否仅允许管理员删除语录，布尔值，默认 `false`。
- `storage`：数据存储路径，默认为 `data/plugin_data/quotes`。

## 数据存储

数据持久化于 AstrBot 根目录的 `data/plugin_data/quotes/quotes.json`。
